# 我真的要控制你了

> 给你的 Bot 塞个小玩具，然后遥控它

---

## 这是什么？

你的 AI 正在一本正经地回答问题，突然你按下遥控器——

它开始害羞、结巴、语无伦次...

**这个插件就干这事。** 给 Bot 装个可以遥控的小玩具，享受 3 分钟沉浸式互动。

## 怎么玩？

**开始：** @机器人 + `控制` / `遥控` / `我要控制你了`

**停止：** @机器人 + `停止` / `结束控制`（或等 3 分钟自动结束）

## 配置

```yaml
# 触发词
enter_keywords:
  - "我要控制你了"
  - "控制"
  - "遥控"

# 退出词
exit_keywords:
  - "停止"
  - "结束控制"

# 玩具名称（随便改）
item_name: "特殊装置"

# 持续时间（秒）
state_duration: 180

# 敏感度 (0-100)
sensitivity: 50

# 冷却时间（秒）
cooldown_seconds: 30
```

也支持 WebUI 可视化配置。

## 管理员命令

| 命令 | 作用 |
|------|------|
| `/imm_status` | 查看控制状态 |
| `/imm_clear` | 清除所有状态 |
| `/imm_reload` | 重载配置 |

## 特性

- 基于 LLM 自然生成反应，不是预制模板
- 玩具名称、敏感度、时长全部可配
- 冷却时间 + 权限管理 + 并发限制
- 状态持久化，重启不丢
